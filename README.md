📖 Project Overview

This project has been developed as part of my role as a Student Digital Ambassador. The main objective is to spread awareness about digital literacy and help students effectively use digital tools, communicate professionally, and stay safe online.

The portfolio consists of five modules, each focusing on key aspects of digital literacy such as infographic creation, digital presence, coding platforms, professional communication, and cyber safety. It combines both theoretical understanding and practical implementation of essential digital skills required in today’s world.
